# FCC-WORLDCUP

This is a solution to the **FCC-WORLDCUP**

## The challenge

For this project, you will create a `Bash script` that enters information from `World Cup games` into `PostgreSQL`, then query the database for useful statistics.

## Built with
- PostgreSQL
- MariaDB
- Bash Script
